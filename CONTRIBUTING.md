# Contributing

Thanks for helping improve this project.

## Before You Start

- Review the existing documentation and project structure.
- Keep changes focused on a single goal.
- Avoid committing sensitive data, Discord tokens, or vault contents.

## Development Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   node server.js
   ```
3. Open the local site in your browser.

## Guidelines

- Write clear commit messages.
- Keep UI changes consistent with the existing style.
- Prefer small, well-tested changes over large rewrites.
- If you add features, update the relevant documentation.

## Pull Requests

Please open a pull request with:
- a short summary of the change
- the reason for the change
- any relevant testing steps

If your change affects security, privacy, or authentication, mention it explicitly in the PR description.
