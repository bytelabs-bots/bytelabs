Frequently Asked Questions

Common questions about the vault, Discord account manager, and project usage.

What is this project?

This app provides a vault interface plus a Discord account manager for connecting stored Discord tokens.

Is my Discord token stored securely?

Tokens are stored in the local vault file and encrypted with the master password you provide. Never share your vault or tokens with others.

Can I use this for personal automation?

Yes, but you should use it responsibly and follow Discord's terms of service and any platform policies.

How do I report a bug?

Open an issue with steps to reproduce the problem. For security issues, please use the private security reporting process instead of a public issue.

How can I contribute?

See the project contribution guide and pull request template in the repository documentation.
